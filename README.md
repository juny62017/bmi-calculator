BMI Calculator

This is a simple BMI calculator website made using HTML, CSS and JavaScript.

I made this to practice basic web development and understand how simple calculations work on a webpage.

## What it does

- Takes height (in cm) and weight (in kg)
- Calculates BMI
- Shows the result with category:
  - Underweight
  - Normal
  - Overweight
  - Obese

## How to use

1. Enter your height in cm  
2. Enter your weight in kg  
3. Click on "Calculate BMI"  
4. Result will be shown below
